This folder contains files required for the iPhone application:

* PhoneGap.plist => Configuration file for PhoneGap
  Edit with a typical text editor to configure specific native features.
* Default.png => Launch image, 320x480 Pixel. 
  Should be kind of screenshot of the empty application. No splash screen.
* Icon.png => Application Icon, 57x57 Pixel
  No shadows, gradients, etc.
* Icon-Small.png => Application Icon for search, 29x29 Pixel
  Small, less details variant of default application icon
* Icon-Store.png => 512x512px
  Richer variant of application icon with additional details. Used for Apple's AppStore.
